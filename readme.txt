Описание файлов в проекте.
_________________________________________________________________________________________

index.php - точка входа в приложение
add.php - скрипт добавления работника
show.php - вывод данных о работниках
functions.php - функции для работы с файлом JSON (чтение, запись)
add_form.html - документ, содержащий форму ввода данных, подключается к скрипту добавления работника
form.js - скрипт для изменения формы в зависимости от типа сотрудника
jquery-3.3.1.min.js - библиотека jQuery
workers.json - файл вида JSON с массивом из работников

Версия PHP: 5.6
Версия Apache: 2.4
